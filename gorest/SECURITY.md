# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| >=1.5.1 | :white_check_mark: |
| 1.4.x   | :white_check_mark: |
| < 1.4   | :x:                |

## Reporting a Vulnerability

Please open a new [issue](https://github.com/piLinux/GoREST/issues/new/choose)
to report a vulnerability.
